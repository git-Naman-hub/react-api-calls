👤 Random Users Explorer (React + CSS)

A clean and modern React application that fetches random user data from an API and displays it using beautifully styled user cards.  
*(Main focus is on learning React concepts, API calls, and component-based UI 😄)*

---

## 🚀 Features

* 🌐 Fetch random users from an external API
* 👤 Display user profile information dynamically
* 🖼️ Show profile image, email, phone number, age, and country
* 🔢 Numbered user cards (UserCard 1, UserCard 2, ...)
* ⚡ Built with React Hooks (`useState`, `useEffect`)
* 🎨 Styled using modern CSS with earthy pastel aesthetics
* ✨ Responsive flexbox card layout

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS3
* Fetch API
