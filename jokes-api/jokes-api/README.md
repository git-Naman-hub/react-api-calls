# 😂 Jokes API Explorer (React + Tailwind)

A fun and modern React application that fetches jokes from an API and displays them in a clean dark-themed UI.
*(Main focus is on learning React concepts and API handling 😄)*

---

## 🚀 Features

* 😂 Fetch jokes from an external API
* 📦 Display multiple jokes dynamically
* 🏷️ Show joke categories when available
* 🎲 Display "Random Joke" when no category exists
* 🔢 Numbered joke cards (Joke 1, Joke 2, ...)
* ⚡ Built with React Hooks (`useState`, `useEffect`)
* 🎨 Styled using Tailwind CSS with dark gradients and glow effects

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* Tailwind CSS (v4)
* Fetch API
